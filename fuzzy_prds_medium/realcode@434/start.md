## Product Requirement Document

We need the full particle-system analysis toolkit surface covered by the tests, not just mean-squared displacement. Implement deterministic utilities for lattice generation, box geometry and coordinate transforms, periodic wrapping, neighbor queries and neighbor-list tables, radial distribution functions, density, cluster identification, nematic and hexatic order, Steinhardt order, and MSD/trajectory analysis. The JSON adapter should route each public feature input to the corresponding analysis and print the exact numeric/list summaries expected by the tests.

One extra detail the team asked about on the MSD side: there are exactly two supported modes, 'window' and 'direct', and 'window' is the default. So if the mode key is absent from the input, 'window' must be assumed. The output here should be a dict with a single key 'msd' whose value is a flat list of floats with exactly one entry per input frame, so the length equals the number of time steps. Also, the first entry at t=0 must always be 0.0 for both modes.

For 'window', the behavior should be the windowed average over all valid time-lag pairs in the trajectory, which is the more statistically robust version we want. Concretely, for each lag time tau, compute the squared displacement over all pairs (t, t+tau) available in the trajectory, average that over all particles, and the result at index tau is the mean over all valid starting frames. This is the same windowed MSD averaging convention described in feature12_mean_squared_displacement.json: for lag tau, average squared displacement over all (t, t+tau) pairs across all particles, equivalent to a time-autocorrelation normalization scheme. Shorter lags will naturally use more statistics.

For 'direct', keep it simpler: for each time step t, the MSD is the mean over all particles of the squared Euclidean distance between the particle's position at time t and its position at time 0 (the first frame). There is no windowing in that mode, only the initial frame is used as the reference.
